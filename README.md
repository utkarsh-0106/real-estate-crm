# 🏠 Real Estate CRM System

A full-stack CRM (Customer Relationship Management) application built to manage real estate leads, properties, and deal pipelines.

---

## 🚀 Features

### 🔹 Lead Management

* Add and track leads
* Status tracking (New → Contacted → Closed)

### 🔹 Property Management

* Add and manage property listings
* View property details

### 🔹 Deal Pipeline

* Create deals linking leads & properties
* Track deal stages:

  * Inquiry
  * Negotiation
  * Closed

### 🔹 Dashboard

* Total leads
* Total properties
* Total deals
* Closed deals

---

## 🛠 Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Frontend:** HTML, CSS, JavaScript

---

## ⚙️ How to Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/real-estate-crm.git
cd real-estate-crm
npm install
npx nodemon server.js
```

Open in browser:

```
http://localhost:5000
```

---

## 🧠 System Flow

1. Capture leads
2. Add properties
3. Create deals linking leads & properties
4. Track deal progress through pipeline
5. Monitor via dashboard
